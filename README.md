BZip2
=====

A fork of [Bzip2](http://www.bzip.org/) to add a CMake build system, without modifying any of the sources. The project can be incorporated into another project or build alone.

To build Bzip2, type:

```bash
mkdir build
cd build
cmake ..
make -j 5
```
